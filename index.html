Process

주제 선정

각자 관심사를 토대로 공공데이터포털과 네이버데이터랩, 영화 관련 기관 홈페이지를 리서치하여 주제를 선정하였습니다.

데이터 수집

부산국제영화제 공식 홈페이지의 데이터를 R 크롤링 기법으로 수집하였습니다.

데이터 정제

서로 다른 국가 표기를 통일하고, 불필요한 데이터를 제거하는 등 R을 이용해 데이터를 분석하기 알맞은 형태로 정제하였습니다.

데이터 분석

R에서 wordcloud, ggplot 등의 패키지를 활용하여 데이터 빈도와 텍스트 분석을 하였습니다.

데이터 시각화

d3.js와 Tableau를 이용해 그래프를 만들었습니다. bootstrap의 소스 코드를 활용해 웹의 기본적인 골격을 잡고, 웹디자인을 진행하였습니다.
